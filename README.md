# Laravel Shopping Cart System

A complete e-commerce shopping cart built with Laravel 10, React (Inertia.js), and Tailwind CSS.

## Features
- User Authentication (Register/Login/Logout)
- Product Browsing with Stock Management
- Shopping Cart (Add, Update, Remove Items)
- Database Persistence per User
- Responsive Design

## Tech Stack
- Laravel 10
- React (Inertia.js)
- Tailwind CSS
- MySQL/SQLite
- Laravel Breeze

## Installation
1. Clone repository
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env`
5. Run `php artisan key:generate`
6. Run `php artisan migrate`
7. Run `php artisan db:seed --class=ProductsSeeder`
8. Run `npm run build`
9. Run `php artisan serve`
